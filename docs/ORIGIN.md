# Origin

This repo extends the `Nonprofit / Foundation Ops` lane in the Kinetic Gain portfolio.

It exists to show a buyer-readable operator surface for:

- donor follow-up
- stewardship evidence
- pledge-risk posture
- executive handoff continuity

The implementation is intentionally synthetic and local-first, but the operating pattern is real: donor trust needs an evidence lane that keeps stewardship promises, finance-safe pledge notes, and campaign posture aligned.
