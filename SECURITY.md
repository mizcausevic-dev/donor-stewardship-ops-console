# Security policy

## Supported versions

Only the latest tagged release is supported for security fixes.

## Reporting a vulnerability

Please open a private security advisory or contact through `kineticgain.com/contact`.

## Scope notes

- All data in this repo is synthetic demonstration data.
- This repo does not process live donor records or regulated submissions.
- Production stewardship systems handling sensitive donor history require formal review, access control, retention controls, and deployment hardening before operational use.
